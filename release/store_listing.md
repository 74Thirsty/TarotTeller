# Store Listing Draft

## Short description
Offline tarot readings with deterministic seeds, daily draw, and local-only history.

## Full description
Tarot Teller gives you a beautiful offline tarot experience:

- Daily Draw (deterministic by date)
- 3-Card Reading (Past/Present/Future)
- Yes/No card pull
- Optional Celtic Cross spread
- Reversal toggle and custom seed input for reproducible readings
- Local history with search
- No account, no ads, no tracking

Everything runs on-device. Your readings stay private.

## Suggested keywords
Tarot, tarot reading, daily tarot, oracle, spiritual, meditation, mindfulness, offline
